#  Password-Manager-RN

A secure and lightweight React Native app to store and manage your passwords locally. Built with React Native (Expo), this app keeps your credentials encrypted and safe â€” without using any cloud service.

---

##  Features

- Add, edit, and delete password entries (website/app, username, password)
-  AES encryption for local storage (vault is encrypted end-to-end)
-  Master password authentication
-  Dark and light theme support
-  One-tap copy to clipboard
-  Encrypted backup , download a copy of stored passwords.

---

##  Tech Stack

- React Native + Expo
- Secure local storage (encrypted SQLite)
- AES-256 encryption 
- Expo router
- TypeScript 

---

##  Getting Started

### Prerequisites

- Node.js and npm or yarn
- Expo CLI (`npm install -g expo-cli`)

```bash
